## Linting & Formatter

- ESLint
- Prettier

## How To Use

To run this application, you'll need Git and Node.js (which comes with yarn) installed on your computer. From your command line:

```bash
# Install dependencies
$ yarn install

# Run the app
$ yarn start

# Run mode dev
$ yarn dev
```
